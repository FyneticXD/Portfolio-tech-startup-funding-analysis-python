# 📊 Analisis Pendanaan Startup Teknologi Menggunakan Python

## 📌 Deskripsi Proyek

Proyek ini bertujuan untuk menganalisis dataset pendanaan startup
teknologi menggunakan Python. Analisis dilakukan untuk memahami pola
investasi berdasarkan perusahaan, wilayah, sektor industri, dan tahap
pendanaan. Proses analisis meliputi *data cleaning*, *exploratory data
analysis (EDA)*, serta visualisasi data guna menghasilkan insight yang
dapat mendukung pengambilan keputusan berbasis data.

------------------------------------------------------------------------

## 🎯 Tujuan Proyek

-   Memahami struktur dan karakteristik dataset.
-   Melakukan pembersihan data (*data cleaning*).
-   Melakukan *Exploratory Data Analysis* (EDA).
-   Membuat visualisasi data.
-   Menghasilkan insight bisnis dari hasil analisis.

------------------------------------------------------------------------

## 🛠️ Tools dan Library

-   Python
-   Pandas
-   Matplotlib
-   Jupyter Notebook

------------------------------------------------------------------------

## 📂 Dataset

**Nama Dataset:** Tech Startup Funding Dataset

Dataset ini memuat informasi mengenai: - Nama perusahaan (Company) -
Website - Wilayah (Region) - Sektor industri (Vertical) - Jumlah
pendanaan (Funding Amount (USD)) - Tahap pendanaan (Funding Stage) -
Tanggal pendanaan (Funding Date)

## 📈 Analisis yang Dilakukan

-   Analisis 10 Startup dengan Pendanaan Terbesar
-   Analisis Total Pendanaan Berdasarkan Wilayah
-   Analisis Pendanaan Berdasarkan Sektor Industri
-   Analisis Distribusi Tahap Pendanaan
-   Analisis Tren Pendanaan Startup Berdasarkan Tahun

------------------------------------------------------------------------

## 💡 Insight Utama

-   Pendanaan startup cenderung terkonsentrasi pada sejumlah kecil
    perusahaan dengan nilai investasi yang sangat tinggi.
-   Beberapa wilayah menjadi pusat investasi startup dan menerima total
    pendanaan yang lebih besar dibandingkan wilayah lainnya.
-   Sektor industri tertentu lebih menarik minat investor sehingga
    memperoleh pendanaan lebih tinggi.
-   Distribusi tahap pendanaan menunjukkan karakteristik perkembangan
    startup pada berbagai fase pertumbuhan.
-   Tren pendanaan mengalami perubahan dari waktu ke waktu sehingga
    dapat digunakan untuk memahami perkembangan ekosistem startup.
